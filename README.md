# cppHusky 的 Rime 自用方案

該方案由 *cppHusky* 基於 [Rime](https://github.com/rime/home) 輸入引擎製作而成。支持以下輸入法：

+ **[五笔 86](#五笔86)（码表源于 [networm 的 86 五笔单字方案](https://github.com/networm/Rime)）**
+ **[倉頡五代](#倉頡五代)（碼表源於 [Jackchows 的倉頡五代補完計劃](https://github.com/Jackchows/Cangjie5)）**
+ **[宇浩·日月](#宇浩·日月)（码表源于 [forfudan 的宇浩输入法](https://github.com/forfudan/yuhao-ime-release)）**

下面陳述各輸入法之配置，以供參閱。

## 全局設定（可以有例外）

全角模式的標點配置相對詳細；半角模式的標點配置比較簡潔。參見默認配置文件 [default.custom.yaml](./default.custom.yaml) 。

只能用 `Ctrl`+`F4` 喚起方案選單。可以用 `Shift` 快捷鍵切換中英文輸入。

翻下頁可以用 `]`；翻上頁可以用 `[`。

所有輸入法均默認使用英文模式。

所有形碼輸入法的詞庫只含單字，不保留詞組。音碼輸入法保留詞組。

形碼輸入法不支持詞組輸入和動態調整詞頻，以期達到最佳盲打效果。

（另：以上方案無一例外，均不允許混合輸入。如，Rime 原本的倉頡輸入法允許混輸拼音，本方案已將該功能取消）

## 五笔86

每页 3 个候选项。

词库包括三万余汉字，含简体字和繁体字，简体字优先。

四码唯一自动上屏；敲击第五码可以顶字上屏。（部分标点符号也有顶字上屏作用）

敲击 `` ` `` 可以通过全拼反查五笔编码。

支持 [仓颉五代符号表](https://zh.wikibooks.org/zh-cn/%E5%80%89%E9%A0%A1%E8%BC%B8%E5%85%A5%E6%B3%95/%E9%80%B2%E9%9A%8E%E7%9F%A5%E8%AD%98#%E4%BA%94%E4%BB%A3%E5%80%89%E9%A0%A1%E7%AC%A6%E8%99%9F%E8%A1%A8) 中的部分 `yyy` 开头符号，以 `zy` 代替 `yyy` 使用。

## 倉頡五代

每頁 3 個候選項。

至少九萬餘漢字，含繁體字和簡化字，繁體字通常優先

五碼唯一自動上屏；敲擊第六碼可以頂字上屏。（部分標點符號也有頂字上屏作用）

敲擊 `` ` `` 可以通過全拼反查倉頡碼。

支授 [五代倉頡符號表](https://zh.wikibooks.org/zh-tw/%E5%80%89%E9%A0%A1%E8%BC%B8%E5%85%A5%E6%B3%95/%E9%80%B2%E9%9A%8E%E7%9F%A5%E8%AD%98#%E4%BA%94%E4%BB%A3%E5%80%89%E9%A0%A1%E7%AC%A6%E8%99%9F%E8%A1%A8)。

碼表中刪去 `重` 碼，改為 `片` 碼。

+ 敲繫 `片戈*` 可以輸入 [表意文字描述字元](https://zh.wikipedia.org/wiki/%E8%A1%A8%E6%84%8F%E6%96%87%E5%AD%97%E6%8F%8F%E8%BF%B0%E5%AD%97%E7%AC%A6)，如 `片戈田一中` 得 `⿸`；
+ 敲繫 `片弓*` 可以輸入計數符號（即畫「正」字），如 `片弓一卜` 得 `𝍴`；
+ 敲繫 `片口*` 可以輸入部首字元，如 `片口弓中` 得 `⻏` 與 `⻖`。
+ 敲繫 `片尸*` 可以輸入筆畫字元，如 `片尸女中` 得 `㇟`。

## 宇浩·日月

每页 3 个候选项。

五码唯一自动上屏；遇到韵码或五码时敲击下一码可顶字上屏。（部分标点符号也有顶字上屏作用）

敲击 `` ` `` 可以通过全拼反查宇码编码。
